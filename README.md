# homebrew-sport-tui

Homebrew tap for [sport-tui](https://github.com/christaylor19/sport-project) — a terminal UI for multi-sport data.

## Install

```bash
brew tap christaylor19/sport-tui
brew install sport-tui
```

## Usage

```bash
# Set your API URL
export SPORT_API_URL=https://your-api-url

# Run
sport-tui
```

## Update

```bash
brew upgrade sport-tui
```
